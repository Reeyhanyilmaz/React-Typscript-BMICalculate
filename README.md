# React-Typscript-BMICalculate
Patika.dev Frontend Web Development Projeleri  "Vücut Kitle Endeksi Hesaplama" çalışması.



## Kullanılan özellikler:
* react.js ve ```TypeScript```
* Stillendirme de ```styled-components``` ve ```SASS```
* Alert uyarılarında ```sweetalert2 ```
* react-router-dom
* localStorage
